## 이미지 확보 방법

### 방법 1: 공식 사이트에서 직접 다운로드
1. 각 브랜드 공식 사이트 접속
2. 모바일상품권 또는 상품권 안내 페이지 이동
3. 상품권 샘플 이미지 또는 대표 이미지 확인
4. 브라우저 개발자도구(F12) > Network 탭에서 이미지 URL 확인
5. 이미지 다운로드

### 방법 2: 직접 구매하여 스크린샷
1. 각 브랜드 모바일상품권 최소 금액 구매
2. 수신한 모바일상품권 이미지 확인
3. PIN 번호 등 민감정보 제거 후 스크린샷
4. 이미지 편집: 개인정보 마스킹, 크기 조정

### 방법 3: 플레이스토어/앱스토어 스크린샷
1. 각 백화점/브랜드 앱 페이지 확인
2. 앱 소개 이미지 중 상품권 관련 이미지 확인
3. 고화질 스크린샷 저장

### 방법 4: 제휴/공식 이미지 요청
1. 각 브랜드 고객센터 또는 마케팅팀 연락
2. 서비스 소개용 이미지 제공 요청
3. 사용 허가 및 저작권 확인

## 이미지 최적화 방법

### macOS 기본 도구 사용
```bash
# 이미지 크기 조정 및 최적화
cd /Users/hasanghyeon/brother_buy/images/real-cards

# 1. PNG를 JPEG로 변환 (AK플라자)
sips -s format jpeg ak.jpg --out ak-temp.jpg
mv ak-temp.jpg ak.jpg

# 2. 이미지 크기 조정 (512x512px)
sips -Z 512 *.jpg

# 3. JPEG 품질 최적화 (품질 80%)
for file in *.jpg; do
  sips -s formatOptions 80 "$file" --out "$file"
done
```

### ImageMagick 사용 (설치 필요)
```bash
# ImageMagick 설치
brew install imagemagick

# 이미지 최적화
cd /Users/hasanghyeon/brother_buy/images/real-cards

# PNG를 JPEG로 변환
convert ak.jpg -quality 85 ak.jpg

# 크기 조정 및 최적화 (한번에)
mogrify -resize 512x512 -quality 85 -format jpg *.jpg

# 특정 파일만 최적화
convert hyundai.jpg -resize 512x512 -quality 85 hyundai-optimized.jpg
```

### 온라인 도구 사용
1. **TinyJPG/TinyPNG**: https://tinyjpg.com/
   - 무손실 압축으로 파일 크기 50-80% 감소
   - 최대 5MB, 최대 20개 파일 무료

2. **Squoosh**: https://squoosh.app/
   - Google 제공, 고급 압축 옵션
   - 브라우저에서 처리 (서버 업로드 없음)

3. **Optimizilla**: https://imagecompressor.com/
   - 압축 전후 비교 가능
   - 품질 조절 가능

## 권장 이미지 스펙

```yaml
dimensions:
  width: 512
  height: 512
  aspect_ratio: "1:1" # 정사각형

format:
  type: JPEG
  quality: 80-90%
  color_mode: RGB

size:
  max_size: 100KB
  target_size: 30-50KB

optimization:
  progressive: true
  strip_metadata: true
  compression: high
```
