# 브라더상품권 매입 전문 사이트

상품권 매입을 전문으로 하는 모바일 최적화 웹사이트입니다.

##  실행 방법

### 1. 로컬 서버 실행

```bash
# Python 내장 서버 사용
cd /Users/hasanghyeon/brother_buy
python3 -m http.server 8080

# 또는 Node.js http-server 사용
npx http-server -p 8080
```

### 2. 브라우저에서 열기

```
http://localhost:8080/index-new.html
```

##  주요 기능

- ✅ **모바일 우선 (Mobile-First)**: 375px ~ 480px 최적화

- ✅ **데스크톱**: 1024px 이상 완벽 지원

## ⚡ JavaScript 기능

1. **모바일 메뉴 토글**: 햄버거 메뉴 클릭 시 열기/닫기
2. **네비게이션 스크롤 효과**: 스크롤 시 배경 변경
3. **활성 메뉴 표시**: 현재 섹션에 따라 메뉴 활성화
4. **부드러운 스크롤**: 앵커 링크 클릭 시 부드러운 이동
5. **갤러리 모달**: 이미지 클릭 시 확대 표시
6. **폼 유효성 검사**: 필수 항목 체크
7. **전화번호 포맷팅**: 자동 하이픈 삽입
8. **스크롤 애니메이션**: Intersection Observer 활용
9. **플로팅 버튼 표시/숨김**: 스크롤 위치에 따라 제어
10. **클릭 투 콜 트래킹**: 전화 버튼 클릭 추적 (분석용)

##  커스터마이징

### 색상 변경
`style-new.css` 파일 상단의 `:root` 변수 수정:

```css
:root {
    --primary-gold: #DAA549;  /* 원하는 색상으로 변경 */
    --dark-bg: #3A3234;
    /* ... */
}
```

### 콘텐츠 수정
`index-new.html` 파일에서 텍스트 내용 직접 수정

### 이미지 교체
