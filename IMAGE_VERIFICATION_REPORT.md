## 추천 이미지 구하기 방법

### 방법 1: 공식 사이트에서 다운로드 (권장)
각 브랜드 공식 홈페이지 → 모바일상품권 소개 페이지 → 샘플 이미지 다운로드

**장점**:
- 공식 디자인
- 저작권 안전
- 고품질 이미지

**단점**:
- 모든 브랜드가 제공하지는 않음

### 방법 2: 구글 이미지 검색 (주의 필요)
```
검색어 예시:
"현대백화점 모바일상품권"
"롯데백화점 모바일상품권 디자인"
"신세계 모바일상품권 샘플"
```

**주의사항**:
- 저작권 확인 필요
- 개인정보 포함 이미지 사용 금지
- 상업적 사용 전 권리 확인

### 방법 3: 직접 구매 후 스크린샷 (가장 정확)
1. 각 브랜드 최소 금액 모바일상품권 구매
2. 수신한 상품권 이미지 확인
3. **PIN 번호/QR코드/바코드 완전 제거**
4. 디자인만 남겨 스크린샷

**장점**:
- 가장 정확한 현재 디자인
- 실제 사용되는 이미지

**주의사항**:
- 개인정보 완전 제거 필수
- 구매 비용 발생

### 방법 4: 앱스토어/플레이스토어 스크린샷
각 백화점 앱 소개 페이지 → 앱 스크린샷 중 상품권 관련 이미지
